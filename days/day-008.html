Day eight — AWS Lambda serverless compute: handlers and runtimes, event triggers from API Gateway to DynamoDB Streams, IAM execution roles, cold starts, concurrency, and how FPO Cloud runs business logic without managing servers.

Sumit Rawal
AWS · FPO Cloud
What I learnt today
✓
Lambda — Serverless compute. Run code without provisioning servers. Pay only for invocations and duration (ms). Core execution layer for FPO Cloud.
✓
Function anatomy — Handler, runtime (Node, Python, etc.), memory (128 MB–10 GB), timeout (up to 15 min). CPU scales with memory.
✓
Triggers — API Gateway, SQS, SNS, EventBridge, DynamoDB Streams, S3 events, scheduled rules. One function, many event sources.
✓
IAM execution role — Lambda assumes a role at runtime. Attach least-privilege policies for DynamoDB, S3, SQS, Secrets Manager, etc.
✓
Concurrency & scaling — Scales automatically per invocation. Reserved concurrency caps a function; provisioned concurrency reduces cold starts.
✓
Layers & env vars — Layers share dependencies across functions. Env vars for config — use Secrets Manager or SSM for secrets, not plain text.
λ Lambda Core — Handler · Runtime · Limits
λ
Handler
Entry point
file.function — e.g. index.handler. Receives event JSON + context object with request id, remaining time.
📦
Deployment package
Zip or container
Zip up to 50 MB direct upload, 250 MB unzipped. Container images up to 10 GB for large dependencies.
/tmp up to 512 MB–10 GB
❄️
Cold start
Init latency
New execution environment spins up on first invoke or after idle. Provisioned concurrency keeps instances warm.
⏱️
Timeout
Max 15 minutes
Default 3 seconds — raise for batch jobs. API Gateway sync invoke max 29 seconds.
🔌 Triggers — How FPO Cloud Invokes Lambda
🌐
API Gateway
REST, HTTP, or WebSocket APIs forward requests to Lambda. Lambda proxy integration passes full request/response.
✦ FPO Cloud API handlers
📬
SQS / SNS
Poll SQS queues or subscribe to SNS topics. Batch size configurable. Failed records go to DLQ after retries.
✦ Async workers & fan-out
⚡
DynamoDB Streams
Stream records trigger Lambda on INSERT/MODIFY/REMOVE. Process change data without polling tables.
✦ Event-driven data pipelines
🛡️ Security & Observability
🔐
Execution role
IAM
Trust policy allows lambda.amazonaws.com. Attach managed policy AWSLambdaBasicExecutionRole for CloudWatch Logs.
✦ One role per function or shared
📊
CloudWatch Logs
LOGS
Automatic log group per function. Metrics: invocations, errors, duration, throttles, concurrent executions.
✦ Debug + alarms in FPO Cloud
🔄
Retry behaviour
Async invokes retry twice on failure. Sync invokes return error to caller. SQS event source retries until visibility timeout expires.
🌍
VPC access
Attach Lambda to VPC subnets to reach private RDS or internal services. Adds ENI cold-start latency — only when needed.
💰
Pricing
Per million requests + GB-seconds (memory × duration). Free tier: 1M requests and 400,000 GB-seconds per month.
